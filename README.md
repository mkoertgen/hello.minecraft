# hello.minecraft

Setting a [Minecraft](https://www.minecraft.net) Server using [Docker](https://www.docker.com/).

## Usage

```console
docker-compose up -d
```

Then fire up your Minecraft Java edition, connect to your server (here: `192.168.2.119`) and enjoy
